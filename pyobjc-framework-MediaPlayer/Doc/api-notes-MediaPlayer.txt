API Notes: MediaPlayer framework
================================

Apple documentation
-------------------

The full API is described in `Apple's documentation`__, both
the C and Objective-C APIs are available (but see the `API Notes`_ below).

.. __: https://developer.apple.com/library/mac/documentation/UserExperience/Conceptual/MediaPlayer/MediaPlayer.html


API Notes
---------

.. note::

   This framework is only available on OSX 10.12 and later and requires a 64-bit binary.
