API notes: FinderSync framework
===============================

Apple Documentation
-------------------

The framework is documented on `Apple's developer website`__

.. __: http://developer.apple.com/library/mac/#documentation/FinderSync/Reference/FinderSyncFrameworkRef/_index.html

API Notes
---------

The entire "FinderSync" framework on MacOSX 10.10 (or later) is available from Python.

.. note::

   The FinderSync framework is only available in 64-bit applications.
