API Notes: Contacts framework
=============================

Apple documentation
-------------------

The full API is described in `Apple's documentation`__, both
the C and Objective-C APIs are available (but see the `API Notes`_ below).

.. __: https://developer.apple.com/reference/contacts/

These bindings are accessed through the ``AppKit`` package (that is, ``import AppKit``).



API Notes
---------

.. note::

   This framework is only available on OSX 10.11 and later and requires a 64-bit binary.
